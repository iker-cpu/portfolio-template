# Portfolio template

Esto es una plantilla de portfolio desarrollado con **HTML y CSS puro**, sin frameworks ni librerías externas.

## Demo

Puedes ver el proyecto desplegado aquí:

https://iker-cpu.github.io/portfolio-template/

## Tecnologías utilizadas

* HTML5
* CSS3

## Estructura del proyecto

```
/
├── index.html
├── styles.css
└── assets/
    ├── github_logo.svg
    └── linkedin_logo.svg
```

## Secciones

* **Sobre mí**
  Breve presentación personal y enfoque profesional.

* **Proyectos**
  Lista de proyectos destacados con tecnologías utilizadas.

* **Experiencia**
  Historial profesional relevante.

* **Educación**
  Formación académica y cursos.

## Características

* Diseño responsive
* Layout con CSS Grid y Flexbox
* Navegación suave (`scroll-behavior`)
* Animaciones sutiles en hover
* Estructura semántica en HTML
* Estilo minimalista y limpio

## Instalación y uso

1. Clona el repositorio:

```bash
git clone https://github.com/iker-cpu/portfolio-template.git
```

2. Abre el archivo:

```bash
index.html
```

No se requieren dependencias ni instalación adicional.
